TCPSERVER-0.0.1
ИНСТРУКЦИЯ
  1. Открыть браузер. В строке адреса набрать адрес http://***.***.***.***:8080, где ***.***.***.*** это IP адрес машины, на которой запущено приложение.
  2. В поле "IP АДРЕС" ввести IP адрес машины, на которой запущено приложение.
  3. В поле "ПОРТ" ввести порт, который будет слушать сервер.
  4. В поле "EOFM БАЙТ" ввести байт, сигнализирующий конец сообщения в шестнадцатеричной системе.
  5. Выбрать формат полученного сообщения - в виде символов ASCII или в виде байтов в шестнадцатеричной системе.
  6. Запустить сервер кнопкой "СТАРТ".
  7. Подключиться к серверу по протоколу TCP (например при помощи Putty в режиме Raw). Отправить на сервер сообщение длиной до 1000 символов.
  8. Сервер отправит в ответ копию вашего сообщения.
  9. Остановить сервер кнопкой "СТОП".
