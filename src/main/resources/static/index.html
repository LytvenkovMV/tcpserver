TCPSERVER-0.0.1
ИНСТРУКЦИЯ
  1. Открыть браузер. В строке адреса набрать адрес http://***.***.***.***:8080, где ***.***.***.*** это IP адрес сервера.
  2. В открывшейся вкладке в поле "IP АДРЕС" ввести IP адрес сервера.
  3. Запустить сервер кнопкой "СТАРТ".
  4. Открыть программу Putty. Выбрать тип подключения "RAW", ввести IP адрес сервера, ввести номер порта 2404 и установить соединение.
  5. Отправить на сервер сообщение любой длины, но не более 1000 символов.
  6. Если все работает правильно, сервер пришлет копию вашего сообщения.
  7. Остановить сервер кнопкой "СТОП".
