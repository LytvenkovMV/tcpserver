TCPSERVER-0.0.1
ИНСТРУКЦИЯ
1. Запустить TCP сервер.
2. Подключиться к TCP серверу по TCP/IP протоколу. Номер порта для подключения 2404.
3. Отправить сообщение, состоящее из 10 символов.
4. Если все работает правильно, сервер пришлет копию вашего сообщения.
5. Для отправки сообщений на сервер можно использовать программу Putty (подключаться в режиме RAW).
