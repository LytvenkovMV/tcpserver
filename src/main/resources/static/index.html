IPSCANNER-1.0
ИНСТРУКЦИЯ
1. Заполните форму ниже. Все поля обязательны.
2. Введите начальный IP адрес. Допустимые значения: 0-255.
3. Выберите префикс маски подсети (диапазон сканирования).
4. Выберите количество потоков.
5. Выберите диск для сохранения результатов сканирования.
6. Нажмите кнопку СТАРТ.
7. Программа найдет все SSL сертификаты по заданным адресам и найдет в них все доменные имена.
8. Список найденных доменных имен будет сохранен в корневом каталоге диска.
