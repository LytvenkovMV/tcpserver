ИНСТРУКЦИЯ
1. Открыть браузер. В строке адреса набрать адрес: http://95.163.185.81:8080
2. В открывшемся интерфейсе запустить TCP сервер кнопкой "СТАРТ".
3. Открыть программу Putty. Выбрать тип подключения "RAW", ввести IP адрес 95.163.185.81, ввести номер порта 2404 и установить соединение.
4. Отправить сообщение, состоящее из 10 символов.
5. Если все работает правильно, сервер пришлет копию вашего сообщения.
6. Остановить TCP сервер кнопкой "СТОП".
